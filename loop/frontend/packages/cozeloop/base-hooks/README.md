# @cozeloop/base-hooks

Hooks for CozeLoop

## Overview

This package is part of the Coze Loop monorepo and provides community features functionality. It includes modal.

## Getting Started

### Installation

Add this package to your `package.json`:

```json
{
  "dependencies": {
    "@cozeloop/base-hooks": "workspace:*"
  }
}
```

Then run:

```bash
rush update
```
