# @cozeloop/stores

Stores For Devops

## Overview

This package is part of the Coze Loop monorepo and provides state management functionality. It includes store.

## Getting Started

### Installation

Add this package to your `package.json`:

```json
{
  "dependencies": {
    "@cozeloop/stores": "workspace:*"
  }
}
```

Then run:

```bash
rush update
```
